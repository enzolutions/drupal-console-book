# debug:rest
Affiche la resource Rest disponible dans l'application

**Usage:**
```
drupal debug:rest [arguments] [options]
rede
```

## Available options
Option | Details
-------|-------------
--authorization | Statut de la ressource rest activé | désactivé

## Available arguments
Argument | Details
---------|-------------
resource-id | ID Rest

## Examples
* Displays rest hooks
```
drupal debug:rest
```
