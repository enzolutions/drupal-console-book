# debug:views
Affiche des informations à propos des vues de l'application

**Usage:**
```
drupal debug:views [arguments] [options]
vde
```

## Available options
Option | Details
-------|-------------
--tag | Tags de la vue
--status | Statut de la vue (Activé|Désactivé)

## Available arguments
Argument | Details
---------|-------------
view-id | Identifiant de la vue

## Examples
* List of views on the site
```
drupal debug:views
```
