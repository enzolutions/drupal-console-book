# list
Liste les commandes

**Usage:**
```
drupal list [arguments] [options]
```

## Available options
Option | Details
-------|-------------
--xml | Afficher la liste sous le format XML
--raw | Afficher la liste sous un format brut
--format | Le format de sortie (txt, xml, json, or md)

## Available arguments
Argument | Details
---------|-------------
namespace | Le nom du namespace
