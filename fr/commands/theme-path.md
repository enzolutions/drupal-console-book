# theme:path
Renvoie le chemin relatif du thème (ou le chemin absolu)

**Usage:**
```
drupal theme:path [arguments] [options]
thp
```

## Available options
Option | Details
-------|-------------
--absolute | Renvoie le chemin absolu du thème

## Available arguments
Argument | Details
---------|-------------
theme | Theme name

## Examples
* Get the path of mytheme
```
drupal theme:path  mytheme
```
