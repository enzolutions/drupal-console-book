# user:password:hash
Génère un hash à partir d'un mot de passe en texte brut.

**Usage:**
```
drupal user:password:hash [arguments]
uph
usph
```

## Available arguments
Argument | Details
---------|-------------
password | Mot(s) de passe en texte brut

## Examples
* Get hash of the word "p455w0rd"
```
drupal user:password:hash  p455w0rd
```
